      Ajuda

      Objetivo

      Site contruido para facilitar acesso a listas em formato .JSON. A ideia é poder iteragir com os itens e a facilidade sem necessitar ne aplicações complexas.

      Como usar?

      Para que funcione, tenha em mãos uma URL com uma extenção .json (https://url.json). Com isso, ira carregar todos os dados em sua tela. Para adicionar você basta clicar em Adicionsr, se estiver tudo certo, você tera o resultado que deseja.

      Como encontrar meu item?

      No menu vai esta disponivel a opção de Pesquisar quando uma lista estiver carregada. Você pode buscar por palavras chaves ou o nome em expecifico.

      Que tipo de lista funciona?

      As lista .JSON seguem um padrão de estrutura com chaves especificas e se seguir as regras irá carregar normalmente.
